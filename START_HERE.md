# 🎮 Полный гайд по запуску CodeNames с мультиплеером

## 📋 Что нужно сделать:

### 1️⃣ Замените старые файлы на новые

**Windows (PowerShell):**
```powershell
cd d:\dev\codenames\src

# Переименуйте старые файлы (бэкап)
mv App.js App.old.js
mv context\GameContext.js context\GameContext.old.js

# Переименуйте новые файлы в рабочие
mv AppNew.js App.js
mv context\GameContextNew.js context\GameContext.js
```

**macOS/Linux (Terminal):**
```bash
cd ~/dev/codenames/src

# Переименуйте старые файлы (бэкап)
mv App.js App.old.js
mv context/GameContext.js context/GameContext.old.js

# Переименуйте новые файлы в рабочие
mv AppNew.js App.js
mv context/GameContextNew.js context/GameContext.js
```

---

### 2️⃣ Установите зависимости

```bash
npm install
```

Это установит:
- `react-router-dom` - для навигации
- `firebase` - уже установлен

---

### 3️⃣ Запустите приложение

```bash
npm start
```

Приложение откроется на `http://localhost:3000`

## 🎯 Как пользоваться:

### А. Создать игру (Хост)

1. **Главная страница** → **"Создать комнату"**
2. Откроется страница с **кодом комнаты** вверху
3. Нажмите **"📋 Копировать ссылку"**
4. Отправьте ссылку друзьям (Telegram, Discord, WhatsApp)
5. **Введите ваше имя** → выберите команду → станьте капитаном
6. Дождитесь других игроков
7. **Выберите тему** слов внизу
8. **"Начать игру"**

---

### Б. Присоединиться к игре

**Вариант 1: По ссылке от друга**
- Кликните на ссылку → автоматически попадёте в комнату
- Введите имя → выберите команду

**Вариант 2: По коду комнаты**
- Главная страница → **"Присоединиться к комнате"**
- Введите код (8 символов) → **"Войти"**

**Вариант 3: Из списка активных**
- Прокрутите вниз на главной
- **"🎯 Активные комнаты"** → нажмите **"Войти"**

---

## 🎮 Процесс игры:

### 1. Лобби

**Минимум для старта:**
- ✅ Хотя бы 1 игрок в красной команде
- ✅ Хотя бы 1 игрок в синей команде
- ✅ Капитан в красной команде (кнопка "Стать капитаном")
- ✅ Капитан в синей команде

**Опционально:**
- Выберите тему слов (по умолчанию: Классические)

### 2. Игровой процесс

**Капитан текущей команды:**
- Видит цвета ВСЕХ карточек
- Вводит **слово-подсказку** и **количество** связанных слов
- Нажимает **"Дать подсказку"**
- ⏱️ Запускается **таймер 60 секунд**

**Игроки команды:**
- Кликают на карточки, пытаясь угадать свои
- Если угадали свою (красную/синюю) - можно продолжать
- Если чужую или нейтральную - ход переходит
- Если **💣 МИНУ** - команда **ПРОИГРЫВАЕТ!**

**Смена хода:**
- Автоматически при ошибке
- Вручную кнопкой **"Закончить ход"**
- Автоматически когда таймер = 0

### 3. Победа

Первая команда, открывшая **все 9 своих слов** - **побеждает!**

## 📱 Тестирование мультиплеера:

### Локальное тестирование:

**Способ 1: Несколько вкладок**
1. Вкладка 1: Создайте комнату → скопируйте ссылку
2. Вкладка 2: Вставьте ссылку → откройте
3. Вкладка 3, 4... (можно до 10+ игроков)

**Способ 2: Обычное + Инкогнито окно**
1. Обычное окно: создайте комнату
2. Ctrl+Shift+N (Chrome) - инкогнито
3. Вставьте ссылку

### Онлайн тестирование (в локальной сети):

1. Узнайте свой IP:
   - Windows: `ipconfig` → IPv4
   - macOS/Linux: `ifconfig` → inet

2. Друзья заходят по адресу:
   ```
   http://ваш-ip:3000/room/код-комнаты
   ```
   Пример: `http://192.168.1.100:3000/room/1234567890`

## 🐛 Решение проблем:

### "Cannot find module 'react-router-dom'"
```bash
npm install react-router-dom
```

### Игроки не синхронизируются
→ Проверьте Firebase Rules (должны быть `.read: true`, `.write: true`)

### Старая версия запускается
```bash
# Убедитесь что переименовали файлы
ls src/App*.js
ls src/context/GameContext*.js

# Должно быть:
# App.js (новый)
# App.old.js (старый)
# GameContext.js (новый)
# GameContext.old.js (старый)
```

### Ошибка "Firebase not initialized"
→ Проверьте `src/utils/firebase.js` - конфигурация должна быть заполнена
